# Secure File Sharing System

A secure full-stack file sharing application built using MongoDB, Express.js, Node.js and JavaScript.

## Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing with bcrypt
- File Upload
- File Download
- File Delete
- MongoDB Database Storage

## Technologies Used

- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt
- Multer
- JavaScript
- HTML
- CSS
